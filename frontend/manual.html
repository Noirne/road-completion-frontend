Issue

Issues are roads or areas that donâ€™t exist in OpenStreetMap. These are the red lines/polygons you find on the map. The goal of this tool is to have a map free of issues.

Edit

If you click on one of the issues a side bar will appear with more information.

If you are logged in, you can you set a status and/or create a note for this specific issue.

Status

The status indicates the current state of an issue.
The status can be set to fixed, false positive and too difficult.

Fixed
Indicates the issue has been fixed on OpenStreetMap

False Positive
When the issue does not need to be fixed in OpenStreetMap. For example an issue that is not really an issue.

Too Difficult
Indicates that the issue is too difficult to solve with the resources that are currently available.

Notes

Notes can be added to a specific issue.
The note added to this issue is connected to OpenStreetMap and will appear at the same location there.
Make sure to add detailed information about that issue to help other mappers find better solutions.

Map styles

Somethimes it is easier to find the location of an issue when looking at a satellite view.
Using the buttons on the bottom of the screen you can toggle between Default and Satellite view.
